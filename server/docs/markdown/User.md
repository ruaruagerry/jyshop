#### 简要描述：

- [获取用户相关信息]

- http://localhost:9001/commcn/api/v1/user.get_user_info

#### 请求方式：

- post

#### 请求参数:

- ` GetUserInfoReq ` : 获取用户信息

|参数名|是否必须|类型|说明|
|:----    |:---|:----- |-----   |


#### 请求示例:
```
{}
```

#### 返回参数说明:

- ` GetUserInfoResp ` : 获取用户信息

|参数名|是否必须|类型|说明|
|:----    |:---|:----- |-----   |
|`isVip` | 否|bool|是否是会员   |
|`balance` | 否|int64|余额   |
|`couponNum` | 否|int64|优惠券数量   |
|`points` | 否|int64|用户积分   |
|`historys` | 否|[]`caoguo.History`|浏览历史   |
